## 🕹️ 游戏玩法说明 (How to Play)
这是一个基于地牢探险的动作冒险游戏。玩家需要控制英雄避开敌人，收集宝箱。

### 操作指南：
* **↑ ↓ ← → (方向键)：** 控制角色移动
* **Space (空格键)：** 挥剑攻击敌人
* **目标：** 收集所有金币并存活下来！

## ⚙️ 全局设置 (Settings)
* **运行帧数 (FPS)：** 60
* **平台：** Windows 10/11
